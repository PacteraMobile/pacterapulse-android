How to import the project?
This project is base on Android Studio, so please make sure you have Android studio installed on your computer.
This project also needs access internet for library dependency, please make sure you have internet access as well.
Open you Android Studio if you are in Quick Start page, select Open an existing Android Studio project and then in file browser select the project folder.
If you have already opened the Android studio, select File->Open then in file browser select the project folder.
That is it.
